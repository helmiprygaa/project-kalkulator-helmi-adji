# kalkulator sederhana 
Project kalkulator sederhana ini berbasis web dan dibuat menggunakan html, css, dan javascript. Kalkulator ini dapat digunakan langsung di browser tanpa perlu aplikasi tambahan.
Kalkulator ini tidak hanya digunakan untuk perhitungan dasar, tetapi juga memiliki beberapa fitur tambahan agar lebih nyaman digunakan.

# fitur
1. Hitung dasar aritmatika
Kalkulator ini dapat digunakan untuk:
tambah (+)
kurang (−)
kali (×)
bagi (÷)
2. Fungsi trigonometri
Tersedia tombol:
sin
cos
tan
Fungsi ini digunakan untuk perhitungan matematika yang berhubungan dengan sudut kayae.
3. Bisa diketik lewat keyboard
Selain menggunakan tombol di layar:
angka dan operator bisa diketik langsung lewat keyboard
tekan enter untuk melihat hasil
tekan backspace untuk menghapus angka
Fitur ini membuat kalkulator lebih cepat dan mudah digunakan.
4. Tombol penting
tombol c digunakan untuk menghapus semua angka
tombol ⌫ (hapus) digunakan untuk menghapus satu angka
tanda kurung ( ) digunakan untuk menghitung rumus yang lebih panjang
5. Fitur history
Kalkulator memiliki ikon history.
digunakan untuk melihat hasil perhitungan sebelumnya
memudahkan pengguna mengecek kembali perhitungan yang sudah dilakukan
6. Tampilan sederhana dan modern
tombol berbentuk bulat
warna nyaman dilihat
tampilan rapi dan tidak membingungkan
Teknologi yang dipakai
html digunakan untuk membuat struktur tampilan
css digunakan untuk mengatur warna dan desain
javascript digunakan untuk mengatur logika perhitungan dan input
Cara menggunakan
buka file index.html di browser
masukkan angka dan operator dengan klik atau keyboard
tekan = atau enter untuk melihat hasil
klik ikon history untuk melihat riwayat perhitungan
